## **User Experience**
When you reach the website it will prompt you to signup or login. This ensure the user is authorize and authenticated before gaining access to the app.

The user can now search for any food in the database. Each item will have an image, ingredients and instructions on how to prep and cook said item. The user also have the option to rate the food, this feedback will be stored in the database to help better the **User Experience**.

# **Future Updates**

The user will also have the option to save and track items.
